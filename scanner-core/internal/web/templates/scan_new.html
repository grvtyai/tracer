Launch Notes

  • Scans start immediately after you submit the configuration. The displayed start time is only informational for now.
  • Local network neighbor discovery is the clearer name for the old Layer 2 option. It only helps on the same broadcast segment and is optional.
  • Reevaluate is off by default. Turn it on only when you want the system to flag uncertain results for later re-checks.
  • Passive mode can stay on auto so Zeek is available when it is useful without forcing passive ingest into every run.
